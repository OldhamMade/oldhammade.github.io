Free-to-use Developer Resources

WorldTimeAPI.org
Plain-text/JSON API which provides the current time for an IP or timezone, targetted at IoT devices
Python-Tool.net
Online tool which can prettify/reformat Python data-structures, and convert them to JSON/YAML/XML
JSON-Tool.net
Online tool which can prettify/reformat JSON and convert it to YAML/XML
XML-Tool.net
Online tool which can prettify/reformat XML
